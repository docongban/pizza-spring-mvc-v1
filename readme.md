# Website quản lí cho cửa hàng bán pizza

### Build Web Service Using

- Front-end: 
	* HTML/CSS
	* Javascrip
	* Bootstrap
- Back-end: 
	* Java Spring boot
	* Spring data jpa
	* Lombok
	* Mysql database
	* Thymeleaf
	* Spring boot devtools

### Version

* JDK 8
* Spring 2.6.6
* Esclip 

### Uer sesstion:

* Home Page
* Product by Category
* Login/Logout
* Register
* Search product by name
* Cart & check-out

### Admin sesstion:

* CRUD category
* CRUD product
* Bill & check bill
